Slope Buddies

A live tracking application built to connect friends at the slopes. Built out all the basics of social media/networking along with a live updating map showing friends locations in relation to yours on the hills.


Using Geofencing practices, I was able to catalog different ski resorts in my PostgreSQL database and implement conditions for users to be notified when a friend in a users network is inside the same resort.
